My Skills

Frontend Development

I enjoy working in the frontend stack the most. This is because I feel that whatever we do in the frontend of an application/software, we are actually making an impact on the end user and also what we do in the frontend is always something tangible. React is my best friend here.

Backend Development

Backend is nearly as fascinating as the frontend. There we have a lot of problem solving and procedural programming. Frameworks take quite a lot of the heavy weight, but I feel less independent in the backend stack. Flask and Django are my current best friends there, but I am determined to master Java and Springboot too, since I believe it will be a very good asset in my skillset.

Mobile App Development

Initially a bit intimidating but later with the power of cross platform frameworks such as React Native (more specifically Expo), Flutter and Dotnet MAUI, mobile app development has the potential to be as fun as frontend development. For "strictly native mobile apps", I have the desire to learn Kotlin and Swift in order to diversify my skillset in mobile app development.

Machine Learning

I started programming with data science and machine learning. That makes me always very interested in the field. Python and its powerful libraries pandas, numpy, matplotlib, seaborn, scikit learn, tensorflow and keras have been my best friends in the field.
